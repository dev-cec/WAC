Description détaillée

La Win32_LogonSession décrit la ou les sessions douverture de session associées à un utilisateur connecté à un système informatique exécutant Windows. documentation : https://learn.microsoft.com/fr-fr/windows/win32/cimwin32prov/win32-logonsession
