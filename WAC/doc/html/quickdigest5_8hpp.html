Description détaillée

A header file for the QuickDigest5 class, a utility for MD5 hashing.

Auteur
Nathanne Isip

This file defines the QuickDigest5 class, which provides functionalities for computing MD5 hashes of strings and files. The class also includes static methods for converting the computed hash to a string representation.

Note
This class is designed as a final class and cannot be inherited.
