Description détaillée

La classe Win32_UserAccountWMI contient des informations sur un compte dutilisateur sur un système dordinateur exécutant Windows. Documentation : https://learn.microsoft.com/fr-fr/windows/win32/cimwin32prov/win32-useraccount
