Description détaillée

Extension block related to CFSFolder and CFileSysItem object. pour les shellbags, si le père est un zip ou assimilé alors les fils ont un format spécial, il faut donc identifier si le père est un zip ne concerne que les fichiers, certains zip sont identifiés comme directory et dans ce cas pas de format special ne concerne que les extensionblock beef0004
