Description détaillée

La classe WMI Win32_TimeZone représente les informations de fuseau horaire dun système informatique exécutant Windows, ce qui inclut les modifications requises pour la transition vers lheure dété. Documentation : https://learn.microsoft.com/fr-fr/windows/win32/cimwin32prov/win32-timezone
